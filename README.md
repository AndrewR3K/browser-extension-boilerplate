# Browser Extension Boilerplate

> Start creating browser extension in minutes with this easy-to-use boilerplate for a Vue3 + Vite boilerplate.

## Features
* Vue 3
* Vite
* TailwindCSS
* Daisy UI
* Packaging Script

## Installation

```bash
$ npm install
```

## Build - Development

```bash
$ npm run dev
```

## Build - Production

```bash
$ npm run build
```

## Package for distribution

```bash
$ npm run pack
```
